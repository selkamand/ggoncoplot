# Simulated Cancer Genome Dataset

An artificial cancer dataset describing mutations found in 9 different
tumour samples. Rows represent mutations.

## Usage

``` r
oncosim
```

## Format

### `oncosim`

A data frame with 143 rows and 3 columns:

- Samples:

  Sample containing mutation in the specified gene

- Genes:

  Mutated gene

- VariantType:

  Type of mutation in gene
